# docker-client-only-binary

This is unofficial docker client only binaries to reduce file size.

## Comparison

file size of docker v1.10.2

|                   | file size  |
| ----------------- | ---------- |
| official binary   | 33,549,384 |
| client only       | 11,881,945 |
| client only + upx |  3,638,920 |
